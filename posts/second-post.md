# Next.js 学习笔记

Next.js 是一个 React 框架，用于构建服务端渲染的 React 应用。

## 为什么选择 Next.js

1. **SSR/SSG** - 支持服务端渲染和静态生成
2. **文件系统路由** - 基于文件结构的路由
3. **API Routes** - 轻松创建 API 端点
4. **自动优化** - 图片优化等功能

  return <h1>Hello World</h1>
